# Rule Based Chatbot

## Overview
This project is a simple Rule-Based Chatbot developed using Python. The chatbot interacts with users by responding to predefined questions and commands using conditional statements.

## Features
- Greets users
- Responds to common questions
- Provides basic assistance
- Handles unknown inputs
- Exits the conversation when the user types "bye"

## Technologies Used
- Python

## How It Works
The chatbot uses `if`, `elif`, and `else` statements to identify user input and generate appropriate responses.

## Supported Commands
- hi
- hello
- how are you
- what is your name
- help
- bye

## Sample Interaction

User: hi  
Bot: Hello! How can I help you?

User: what is your name  
Bot: My name is Codmetric Bot.

User: bye  
Bot: Goodbye! Have a nice day.

## How to Run

1. Open the project folder in VS Code.
2. Open the terminal.
3. Run the following command:

```bash
python chatbot.py
```
